# 실행 할 시 주의할 점
코드 내의 서버 주소가 .env 파일을 통한 환경변수로 지정되어 있기에 백엔드 서버를 실행하는 서버 컴퓨터의 IP를 환경변수로 지정해야만 함</br>
예시 코드)
```env
//.env 파일
REACT_APP_SERVER_IP="여기에 서버 IP 입력"

```
